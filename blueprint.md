# Blueprint: What to Eat for Dinner Today? (오늘 저녁 뭐 먹지?)

## Overview
A simple, modern web application that helps users decide what to eat for dinner. When the user clicks the "Get Menu Recommendation" (메뉴 추천받기) button, the application displays a randomly selected list of 5 food options.

## Features
*   **Menu Recommendation:** Randomly selects 5 distinct food items from a predefined list.
*   **Modern UI:** A clean, responsive design with elegant typography, soft shadows, and a vibrant color palette, adhering to modern web design principles.
*   **Interactive Elements:** Button interactions (hover, active states) with smooth transitions.
*   **Accessibility:** Semantic HTML and appropriate contrast for readability.

## Architecture
*   `index.html`: Main structure of the application.
*   `style.css`: Modern styling using CSS variables, flexbox/grid, and modern color spaces.
*   `main.js`: Vanilla JavaScript handling the random selection logic and DOM updates.
